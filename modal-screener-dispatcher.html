Screener Dispatcher
How to use: Create a new screener by pressing "Add new screener". When the screener is ready, you can export its results to Sheets or perform an EOD Bulk request and Fundamental data bulk request. The result will also be exported to your sheets in a new page. Please allow us some time to perform the Bulk request - it might involve a lot of data. The window will be closed automatically once the data is ready.

Please note that one Screener request consumes 5 API calls.
