Live (Delayed) Dispatcher
How to use: Create a new Live stream by pressing "Add live stream". When the stream is ready, by switching it "On", a live (delayed) data will be added to a new sheet. Data is delivered in 1-minute interval and with a 15-20 minute delay.

Important. If the exchange is closed, the request will return the last traded price.
